# Juquyz

Nuestro producto es un videojuego de preguntas y respuestas sobre la cultura y hechos históricos de la provincia de Jujuy apuntada al público joven y a los turistas. 
 
Evaluamos que esta propuesta es una buena solución porque implementamos un espacio inmersivo y llamativo para invitar a nuestro público objetivo a aprender sobre Jujuy.
 
La tecnología seleccionada para el proyecto es el motor de videojuegos llamado Unity el cual permite crear un entorno virtual basado en un salón de clases. Este videojuego funciona como un examen o quiz, el cual te dará una pregunta y 3 respuestas posibles el cual deberás seleccionar, se implementa un sistema de puntaje, un sistema de tiempo y un entorno de tercera dimensión. 

Nos inspiramos en proyectos que intentan abordar o mostrar la cultura jujeña desde un enfoque moderno. Por ejemplo el comercial televisivo: “JUJUY, UN DESTINO DE PELÍCULA”, que recibió buena recepción y reconocimiento tanto público como de la crítica. Esto demuestra que este tipo de propuestas son necesarias, solicitadas y valoradas(Visit Jujuy, 2022).
